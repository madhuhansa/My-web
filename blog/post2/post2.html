Before diving into any data analysis, it's crucial to thoroughly understand your dataset. This guide walks through the essential checks you should perform when first examining a new dataset.

1. Dataset Shape

The shape tells you how many rows (observations) and columns (features) your dataset contains.

df.shape

Always Remember to view the shape to understand how shape change. this helps you to make sure you do not lose important data by any accident when you do data cleaning, outliers removiers or feature engineering

2. Column Information

Examine column names, data types, and basic statistics:

# View all the columns
df.columns 

# View column names and data types
df.info()

# Get statistical summary
df.describe()

# For categorical columns
df.describe(include='all')

Numerical columns: Check min, max, mean values
Categorical columns: Examine unique values
Datetime columns: Verify date ranges

3. Missing Values

Identify columns with null values and decide how to handle them in data Cleaning Step

df.isnull().sum()

4. Duplicate Records

Check for and handle duplicate rows:

# Check for duplicates
df.duplicated().sum()  

# Remove duplicates
df = df.drop_duplicates()

# Check for duplicates in specific columns
df['column_name'].duplicated().sum()

Consider whether duplicates represent valid data (like multiple transactions from same customer) or need removal.

5. Object Data Values

Analyze text/categorical columns:

# Get a summary of the DataFrame
df.info()

# Count unique values in a categorical column
df['colname'].value_counts()

#describe table for object data
df.describe(include="object")

Understand object columns is very important before come to data cleaning step. Check for inconsistencies, typos, or unexpected values in categorical columns. This is crucial for ensuring data quality

Understanding these fundamental aspects of your dataset will save hours of troubleshooting later. Always perform these checks before any analysis or modeling. Remember to document your findings - what you learn during exploration will inform every subsequent step in your data workflow.
